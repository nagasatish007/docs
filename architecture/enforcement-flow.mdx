# Enforcement Flow

```mermaid
flowchart TD
  A[Evaluate policy] --> B{Mode}
  B -->|REPORT_ONLY| C[ALLOW + log]
  B -->|MONITOR| D[ALLOW + log violations]
  B -->|ENFORCE| E{Violation?}
  E -->|Yes| F[DENY/REQUIRE_APPROVAL + throw PolicyViolationError]
  E -->|No| G[ALLOW]
```
