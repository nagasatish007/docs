# TealTiger SDK (v1.1.0)

TealTiger is a **developer-first AI security and cost governance SDK**. It provides:

- **Policy rollout modes**: ENFORCE / MONITOR / REPORT_ONLY
- **Deterministic Decision contract** (action + reason codes + risk score)
- **Audit schema with redaction-by-default**
- **Correlation IDs & traceability** across SDK components
